Multiprocessing
===============

.. code:: python

    import pyblaze.multiprocessing as xmp

The multiprocessing module provides utilities to easily distribute work across multiple processes.
Its sole purpose is to speed up computation on a single machine when it is too costly to switch to
a faster language than Python.
